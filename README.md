## ✨ Overview

**Tiles Tapp** is a modern arcade reflex game inspired by classic tile tapping mechanics.

The objective is simple:

> Tap the falling tiles before they escape the screen.

But as your score increases, speed rises, pressure builds, and focus becomes everything.

Built as a polished SwiftUI project with scalable architecture and smooth animations.

### 🎮 Gameplay

## 🏗 Architecture

Built using **MVVM Architecture** for maintainability and scalability.

```plaintext
Tiles_Tapp
│
├── Models
│   └── Tile.swift
│
├── ViewModels
│   ├── GameViewModel.swift
│   └── StatsViewModel.swift
│
├── Views
│   ├── HomeView.swift
│   ├── GameView.swift
│   ├── StatsView.swift
│   └── TileView.swift
│
├── Components
│   └── MenuButton.swift
│
├── Services
│
└── Utilities
```
